# EBS™ Community

**快速应用AI，无需自有IT团队。**

EBS Community 是 EBS 企业AI操作系统的开源体验入口，专为无自有IT团队的行业企业及个人创业者设计。

**[ebs.top](https://ebs.top)**

## 核心能力

- **多模型接入支持** —— 配置密钥即可开始对话，无需关心底层接入细节
- **基础图像生成能力** —— 用于快速体验与能力验证
- **可视化交互界面** —— 开箱即用的 Web 界面，本地几分钟即可跑通

## 你将获得什么

克隆本仓库，你拿到的不是代码片段，而是一个**可直接运行的完整AI应用**：

| 交付物 | 说明 |
| --- | --- |
| 可运行的 Web 应用 | 一条命令安装，一条命令启动，几分钟内在本地跑通 |
| 多模型对话能力 | 统一的对话界面，多家模型服务按需切换，切换不改代码 |
| 图像生成体验 | 内置图像生成演示，直观感受多模态能力 |
| 可通读的代码结构 | 单一代码库，模块边界清晰，可以从入口读到底层 |
| Apache 2.0 许可 | 自由使用、修改、二次开发，派生作品保留署名信息即可 |

零账户、零注册、零数据库。密钥仅从环境变量读取，除您自行配置的服务之外，不会向任何地方发送任何数据——**你的评估过程完全在你自己的机器上完成**。

## 你可以用它做什么

- **评估模型效果** —— 在同一界面下切换不同模型，用你自己的业务问题实测对比，几分钟得出结论
- **给客户或团队做演示** —— 本地起服务即可展示AI对话与图像生成能力，无需申请环境、无需报备采购
- **验证业务想法** —— 把你的场景假设放进真实的对话流中快速试错，在投入开发之前确认方向
- **作为二次开发起点** —— Apache 2.0 协议下自由改造：换界面、加功能、接入你自己的业务逻辑
- **学习AI应用架构** —— 一个刻意保持小巧的代码库，是理解"AI应用如何组织"的可运行教材

## 它为你带来的价值

**省时间**：自行搭建一个多模型对话应用，通常意味着数天到数周的选型、接入与调试。本仓库把这个过程压缩到几分钟。

**零决策成本**：Apache 2.0 协议、无账户、无绑定。试了不合适，删掉即可，不留任何痕迹。

**透明可控**：所有代码可读，所有请求可查。密钥在你手里，数据在你机器上——评估AI能力不需要先交出信任。

**架构预览**：本仓库与 EBS 企业AI操作系统采用同源架构。你在这里看到的工程组织方式，就是进阶版本的真实底座——**先验证架构，再决定投入**。

## 快速开始

需要 Node 22 及以上版本。

```bash
npm install
cp .env.example .env.local   # 按需配置所需服务密钥
npm run dev
```

打开 <http://localhost:3000>。未配置任何密钥时应用仍可启动；配置密钥后即可开启对应功能。

密钥仅从环境变量读取。本仓库不含数据库，除您自行配置的服务之外，不会向任何地方发送任何数据。

## 痛点与解决方案

**痛点**：企业拥有深厚的行业经验，却没有AI技术团队，业务知识难以转化为AI能力。

**方案**：EBS 系统将技术复杂性封装于系统之中，用户仅需关注业务本身。
